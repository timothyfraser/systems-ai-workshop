# 🎴 ACTIVITY · Private Coding Agent (Cline)

> ⏱️ **5 minutes** · 🎯 **Let a local agent refactor your code — a real coding agent, with nothing leaving your laptop.**

**Do this:** *(assumes Cline is set up — see [Cline setup](ACTIVITY_cline_local.md))*

1. Open **Cline** in Cursor (or VS Code / Positron) and confirm its model is your local Ollama one.
2. Open a short script, and give Cline a scoped task:
   ```
   Add type hints and a one-line docstring to each function in this file.
   Show me the diff before applying. Keep behavior identical.
   ```
3. Review the diff and approve. (Keep **auto-approve off**.)

✅ **Done when:** a model running on `localhost` edited your file — no account, no cloud. Turn off Wi-Fi and it still works. 🔌🚫
