# 📌 ACTIVITY

## Create HTML Documents (Posters, Flyers, Memos)

🕒 *Estimated Time: 20 minutes*

---

## ✅ Your Task

Use Claude Desktop to generate a structured **HTML document** — a course flyer, a research poster, or a departmental memo — that you can open in a browser and print to PDF. HTML gives you precise control over layout without wrestling with Word.

### 🧱 Stage 1: Describe what you want

- [ ] Open Claude Desktop and start a new chat.
- [ ] Describe your document: purpose, audience, sections, any text to include, and a visual tone (e.g., "clean academic," "bold and colorful").
- [ ] Mention the format: *"Produce a single self-contained HTML file with inline CSS, sized for US Letter, print-friendly."*

Example prompt:

> Create a single self-contained HTML file for a one-page course flyer for "SYSEN 5999: Systems & AI." Include a title banner, a 3-sentence description, a bulleted list of 4 topics, meeting time/location placeholders, and my name. Clean academic style, US Letter, print-friendly with inline CSS.

### 🧱 Stage 2: Use the Skill

- [ ] Open this repo's folder so the [`html-poster`](../.claude/skills/html-poster/SKILL.md) Skill is available, then ask Claude to use it. The Skill enforces self-contained, printable output and sensible typography.

### 🧱 Stage 3: Preview and refine

- [ ] Save Claude's output as `flyer.html` and open it in a browser.
- [ ] Iterate: *"make the banner navy," "tighten the spacing," "add a QR code placeholder."*
- [ ] Print to PDF (Ctrl/Cmd+P → Save as PDF) when you're happy.

---

## 💡 Why HTML

- **Portable:** one file, opens anywhere, no software to install.
- **Precise:** CSS gives you exact control over layout and print.
- **Iterable:** tweaking is a sentence, not a drag-and-drop fight.

---

## 📤 Try It

Make a flyer for a real upcoming talk or class, refine it in three rounds, and print it to PDF.
