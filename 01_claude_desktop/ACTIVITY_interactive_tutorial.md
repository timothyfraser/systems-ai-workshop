# 🎴 ACTIVITY · Interactive Tutorial

> ⏱️ **5 minutes** · 🎯 **Turn one concept you teach into a tappable, phone-friendly mini-tutorial.**

**Do this:** open Claude Desktop on this repo (so the `interactive-toolmaker` skill is available) and paste the prompt below. Swap in a concept from your own course, or use the network-science example as-is.

```
Use the interactive-toolmaker skill to build a one-page interactive tutorial.
Big-picture question: "Why does an outbreak spread faster in some networks than others?"
Concept network: 7 nodes — node, edge, degree, hub, path length, clustering, super-spreader —
with edges showing how they relate.
Include 3 learning checks, each with a tap-to-reveal hint and a tap-to-reveal answer.
Give me one self-contained .html file.
```

✅ **Done when:** you open the saved `.html`, drag the network around, and tap to reveal an answer — on your phone too.

💬 **Discuss:** what would make this reusable across a whole course — a shared color scheme, a fixed template, a saved skill? (That's exactly what the skill does.)
