# 🎴 ACTIVITY · Cursor vs. Claude

> ⏱️ **5 minutes** · 🎯 **Run the same small task in both tools and feel which fits which job.**

**Do this:**

1. In **Cursor**, open `messy.py` (from the [Refactor card](ACTIVITY_refactor.md)) and have it clean the code up in place.
2. In **Claude Desktop**, paste the same code and ask:
   ```
   What would a code reviewer criticize about this, and what's the cleaner structure? Don't rewrite it — explain.
   ```

✅ **Done when:** you've seen Cursor *edit* and Claude *reason* about the same snippet.

💬 **Discuss:** which felt better for hands-on editing, and which for thinking it through? When would you chain them — design in Claude, build in Cursor, review in Claude?
