# 🎴 ACTIVITY · Add Live Docs

> ⏱️ **5 minutes** · 🎯 **Point Cursor at a library's real documentation so it stops guessing function names.**

**Do this:**

1. Open **Settings** (`Ctrl/Cmd+,`), search `docs`, open **Indexing & Docs → Docs**, click **Add new doc**, and add one library you use — for example:
   - **pandas** → `https://pandas.pydata.org/docs/`
2. Wait for it to index (~a minute), open a chat (`Ctrl/Cmd+L`), and ask:
   ```
   @Docs pandas Write code to read a CSV and return the mean of each numeric column.
   ```

✅ **Done when:** the answer cites the real docs and uses functions that actually exist.

💬 **Discuss:** how much of "AI writes wrong code" is really "AI had stale or no docs"? What libraries would you index first?
