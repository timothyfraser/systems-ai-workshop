> ⏱️ **5 minutes** · 🎯 **Generate a clean, printable one-page flyer — describe it, don't design it.**

**Do this:** paste the prompt below into Claude Desktop. Edit the title and details to match a real talk or class of yours.

```
Use the html-poster skill to make a one-page US-Letter flyer for a talk titled
"Networks & Resilience." Include a 3-sentence abstract, speaker name + time/room
```
