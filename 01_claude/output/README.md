# 📦 Example Outputs

Real artifacts produced by the Claude exercises — so you can see what each tool actually makes before you run it yourself.

| File | From the exercise | What it is |
|------|-------------------|------------|
| [`interactive_tutorial.html`](interactive_tutorial.html) | 🧪 Interactive Tutorial (`interactive-toolmaker`) | a self-contained interactive concept-network mini-lesson — open it in a browser |
| [`poster.html`](poster.html) | 🖼️ Poster (`html-poster`) | a printable one-page flyer — open and print to PDF |
| [`CODEBOOK.md`](CODEBOOK.md) | 🧬 Sub-agents → Codebook (`column-documenter`) | a data dictionary built one-column-at-a-time from [`../examples/social_infra.csv`](../examples/social_infra.csv) |
| [`mentions.csv`](mentions.csv) | 🗞️ Monthly Routine (`scholar-mentions-log`) | an *illustrative* mentions log (placeholder rows — real runs pull live web results) |

> The HTML files are best viewed by opening them in a browser. `mentions.csv` is a synthetic sample for illustration; a real run fills it with actual web findings and never invents URLs.
