# 📌 ACTIVITY

## Install Git and a Terminal

🕒 *Estimated Time: 10 minutes*

---

## ✅ Your Task

Install **Git**. On Windows this also gives you **Git Bash**, a Unix-like terminal we'll use for every command in this workshop. On macOS and Linux you already have a terminal; you just need Git.

### 🧱 Stage 1: Install Git

- [ ] Go to the 🔗 [Git download page](https://git-scm.com/downloads)
- [ ] Download Git for your operating system
- [ ] Run the installer and accept the defaults (the defaults are fine)
- [ ] **Windows:** the installer includes **Git Bash**. Open it afterward from the Start menu
- [ ] **macOS:** open the built-in **Terminal** app (or install Git via [Homebrew](https://brew.sh/): `brew install git`)

> **New to the command line?** You only need a handful of commands: `cd` (change folder), `ls` (list files), and `git`. We'll walk through them on the day.

### 🧱 Stage 2: Verify

- [ ] Open your terminal (**Git Bash** on Windows, **Terminal** on macOS)
- [ ] Run:
  ```bash
  git --version
  ```
- [ ] You should see something like `git version 2.45.0`

---

## 💡 Why we need it

Every other tool in this workshop assumes you have a terminal and Git. Git Bash also gives Windows users the same `bash` commands the rest of us use, so the instructions work the same on every machine.
