# 🔧 SETUP · Git + a Terminal

> ⏱️ ~10 min. On Windows, Git also installs **Git Bash**, the terminal we use for every command. macOS/Linux already have a terminal.

1. Install Git from [git-scm.com/downloads](https://git-scm.com/downloads) — accept the defaults.
   - **Windows:** open **Git Bash** from the Start menu afterward.
   - **macOS:** use the built-in **Terminal** (or `brew install git`).
2. Verify:
   ```bash
   git --version
   ```
   You should see something like `git version 2.45.0`.

> New to the command line? You only need `cd`, `ls`, and `git` — we'll walk through them on the day.
