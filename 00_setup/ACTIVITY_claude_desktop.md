## Install Claude Desktop

🕒 *Estimated Time: 10 minutes*

---

## ✅ Your Task

Install **Claude Desktop**, Anthropic's desktop app for **Claude**. We use it for conversational analysis, document creation, **Skills**, **routines**, and as the home of **Claude Code** (a command-line coding agent).

### 🧱 Stage 1: Install the app

- [ ] Go to 🔗 [claude.ai/download](https://claude.ai/download)
- [ ] Download the app for **macOS** or **Windows**
- [ ] Run the installer and open **Claude Desktop**

### 🧱 Stage 2: Sign in

- [ ] Sign in with your Anthropic account (create a free one if you don't have it)
- [ ] **Cornell note:** if your department has a Claude for Work / Team plan, sign in with your Cornell email so your usage is covered. Otherwise a free or Pro account works for the workshop.

### 🧱 Stage 3: Verify Skills load

This repo ships a [`.claude/`](../.claude/) folder with ready-made **Skills**. To use them, you'll open this repo's folder during the workshop. For now, just confirm the app opens and you can start a new chat.

- [ ] Open a new chat and type *"Hello"* — you should get a reply
- [ ] Find **Settings** and locate the **Skills** / **Capabilities** area (we'll configure it together)

---

## 💡 What's a "Skill"?

A **Skill** is a small folder of instructions (a `SKILL.md` file) that teaches Claude how to do a specific job well — like building an HTML poster or red-teaming a paper. This repo's [`.claude/skills/`](../.claude/skills/) folder has several you'll use on the day. You don't have to write any now.
